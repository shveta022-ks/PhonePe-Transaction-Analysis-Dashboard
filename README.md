# 📊 PhonePe Transaction Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard built to analyze PhonePe transaction data and uncover meaningful business insights. The dashboard provides a comprehensive view of transaction performance, payment success rates, customer demographics, service utilization, and monthly trends through interactive visualizations.

The objective was to transform raw transactional data into an easy-to-understand dashboard that supports data-driven decision-making.

---

## 🎯 Business Objectives

- Monitor transaction performance over time.
- Analyze payment success rates.
- Identify the most used PhonePe services.
- Understand customer demographics.
- Compare weekday and weekend transaction activity.
- Build an interactive dashboard for business users.

## 📂 Dataset

The dataset required minimal preprocessing.

### Power Query Transformations
- Corrected Date data type.
- Standardized Payment Status values.
- Verified column formats for accurate analysis.

---

## 📈 Data Modeling

Created a separate Date Table and established relationships between tables to enable accurate time-based analysis and DAX calculations.

---

## 📊 DAX Measures

Developed custom measures including:

- Total Transactions
- Total Transaction Value
- Unique Users
- Successful Transaction Rate
- Previous Month Transactions
- Previous Month Transaction Value
- Month-over-Month (MoM) Transaction Growth
- Month-over-Month (MoM) Transaction Value Growth

## 📌 Key Insights

- Money Transfer contributes the highest transaction value.
- Gen X users represent the largest customer segment.
- Weekdays account for the majority of transactions.
- Transaction performance shows positive Month-over-Month growth.
- Payment success rate remains consistently high.
